# **App Name**: Ace Academy

## Core Features:

- Hero Section: A dynamic hero section with a strong message about poker as a mind sport, and clear calls to action to explore the site.
- Club Information: Display details about the poker club, including upcoming tournaments, competition results, and membership information.
- Community Forum: Dedicated space for members to interact, share strategies, discuss game analysis, and team up for competitions.
- Curriculum Showcase: Detailed presentation of the poker school's curriculum, highlighting key areas such as strategy, discipline, and bankroll management.

## Style Guidelines:

- Primary color: Gold (#D4AF37) for a premium and elegant feel.
- Background color: Black (#000000) for a high-end and disciplined aesthetic, desaturated gold (HSL 51, 23%, 9%) for the landing section.
- Accent color: White (#FFFFFF) to create strong contrast and a clean, legible design, analogous light yellow-gold (HSL 61, 67%, 90%).
- Font pairing: 'Poppins' (sans-serif) for headlines, providing a geometric and modern feel, combined with 'PT Sans' (sans-serif) for body text to ensure readability.
- Responsive layout designed with a mobile-first approach to ensure seamless adaptation across various devices and screen sizes. Incorporate a hamburger menu for mobile navigation.
- Crisp, minimalist icons for main menu items and calls to action.
- Subtle animations to enhance user engagement. Examples: button hover effects, content fade-ins, or animated transitions between sections.
